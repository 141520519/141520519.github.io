### الخصم = القيمة الإسمية - القيمة الحالية

#### الرموز المستخدمة: 
- القيمة الإسمية (ق س)  
- معدل الخصم (ع)  
- مدة الخصم (ن)  
- الخصم (B)  
- القيمة الحالية (ح)  

#### القاعدة:
- عندما تكون **المدة بالأيام**:

B = (ق س x ع x ن) / 36000

- عندما تكون **المدة بالأشهر**:

B = ق س x (ع/100) x (ن/12) B = (ق س x ع x ن) / 1200

- عندما تكون **المدة بالسنوات**:

B = (ق س x ع x ن) / 100
